# 港股ETF板块轮动策略

基于20日动量的港股板块ETF轮动策略，支持回测和IBKR实盘交易。

## 策略概述

使用20日动量指标对港股ETF池进行评分，选择Top N个ETF等权持仓，定期调仓。

## ETF池 (15个)

覆盖港股主要板块：
- 指数: 2800.HK (盈富基金), 2828.HK (恒生H股)
- 科技: 3033.HK (恒生科技), 3032.HK (恒生互联网), 3067.HK (资讯科技)
- 金融: 2829.HK (金融), 2827.HK (银行)
- 消费: 3036.HK (恒生消费), 2826.HK (消费)
- 医疗: 3069.HK (恒生医疗)
- 能源: 3097.HK (恒生能源)
- 地产: 2832.HK (地产)
- 工业: 2825.HK (工业)
- 材料: 3009.HK (原材料)
- 公用事业: 3008.HK (公用事业)

## 使用方法

### 1. 安装依赖
```bash
pip install pandas numpy yfinance optuna ib_insync
```

### 2. 获取数据
```bash
cd hk/data
python fetch_data.py
```

### 3. 参数优化
```bash
cd hk
python optimize.py
```

### 4. 实盘交易

配置IBKR连接后运行实盘交易脚本（需先完成参数优化）。

## 策略逻辑

1. 每日计算所有ETF的20日动量评分
2. 选择评分最高的Top N个ETF等权持仓
3. 每隔N天根据最新评分调仓
4. 手续费: 单边0.005%

## 市场特点

- 交易时间: 9:30-12:00, 13:00-16:00 (港股时间)
- 与A股相关性高，但有独特科技股标的
- 流动性: 主流ETF流动性良好
- 货币: 港币计价

## 风险提示

- 历史回测不代表未来表现
- 港股与A股相关性较高，分散化效果有限
- 需要考虑汇率风险（如使用美元账户）
- 建议与A股、美股策略配合使用

# 港股ETF板块轮动策略

基于20日动量的港股板块ETF轮动策略，支持回测和IBKR实盘交易。

## 策略概述

使用20日动量指标对港股ETF池进行评分，选择Top N个ETF等权持仓，定期调仓。

## ETF池 (15个)

覆盖港股主要板块：
- 指数: 2800.HK (盈富基金), 2828.HK (恒生H股)
- 科技: 3033.HK (恒生科技), 3032.HK (恒生互联网), 3067.HK (资讯科技)
- 金融: 2829.HK (金融), 2827.HK (银行)
- 消费: 3036.HK (恒生消费), 2826.HK (消费)
- 医疗: 3069.HK (恒生医疗)
- 能源: 3097.HK (恒生能源)
- 地产: 2832.HK (地产)
- 工业: 2825.HK (工业)

## 进阶策略：灵敏宏观 + 阶梯仓位 (Sensitive Macro Control)

港股作为离岸市场，极度敏感于美元流动性和人民币汇率。系统引入了**“灵敏判定”**逻辑，以平衡反弹力度与下行风险。

### 核心逻辑：斜率判定与抄底机制
1.  **汇率斜率控仓 (FX Slope)**: 监测 **离岸人民币 (USD/CNH)** 的 5 日变动率。
    - **加速贬值 (Slope > 0.5%)**: 系统性风险预警，总仓位强制压低至 **30%**。
    - **平稳/升值期**: 允许维持 **80%-100% 仓位**，捕捉港股的高弹性反弹。
2.  **利率定风格**: 监测 **10年期美债收益率 (US10Y)**。
    - `利率下行`: 全力增强**恒生科技 (3033.HK)** 等高弹性板块。
    - `利率上行`: 自动转向**港股红利 (3110.HK)** 或银行板块避险。

### 历史回测对比 (2023 - 2025)
*测试条件：Top 2 集中持仓, 10日调仓, 含 0.05% 单边滑点*

| 指标 | 纯技术面动能 (Baseline) | **灵敏宏观版 (HK-Macro)** |
| :--- | :--- | :--- |
| **累计收益** | **141.77%** | 118.54% |
| **最大回撤** | -24.05% | **-18.64%** (显著降低) |
| **夏普比率** | 1.18 | **1.24** (风险收益比最优) |

**研究结论**：在港股这种极其不稳定的市场，**“活下来”比“赚得多”更重要**。宏观版虽然在爆发式反弹（如924行情）中由于汇率延迟确认略有踏空，但在长达两年的震荡市中，通过有效的回撤控制，实现了更高质量的净值增长。

## 市场特点

- 交易时间: 9:30-12:00, 13:00-16:00 (港股时间)
- 与A股相关性高，但有独特科技股标的
- 流动性: 主流ETF流动性良好
- 货币: 港币计价

## 风险提示

- 历史回测不代表未来表现
- 港股与A股相关性较高，分散化效果有限
- 需要考虑汇率风险（如使用美元账户）
- 建议与A股、美股策略配合使用

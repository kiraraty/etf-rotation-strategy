## 进阶策略：技术选股 + 宏观风控 (Macro Position Sizing)

针对美股市场的强 Beta 属性，系统引入了**“动态仓位管理”**逻辑，不再死板地满仓轮动，而是根据流动性闸门决定投入金额。

### 核心逻辑：流动性双因子风控
1.  **利率风险 (Rate Risk)**: 监测 **10年期美债收益率 (US10Y)**。
    - `US10Y > MA20`: 利率上行，分母端受压。
2.  **汇率风险 (FX Risk)**: 监测 **美元指数 (DXY)**。
    - `DXY > MA20`: 美元走强，收割全球流动性。

### 仓位阶梯 (Position Ladder)
- **双利好 (均线下行)**: **100% 仓位**，全速追逐动能。
- **单风险 (其中之一上行)**: **70% 仓位**，适度防御。
- **双风险 (共振大跌预警)**: **40% 仓位**，极度保守，保住本金。

### 进阶对比回测 (2021 - 2025)
*测试条件：Top 2 集中持仓, 10日调仓, 含 0.05% 单边滑点*

| 指标 | 纯技术面动能 (100% 仓位) | **进阶版 (动态仓位控制)** |
| :--- | :--- | :--- |
| **累计收益** | 70.44% | **102.03%** |
| **最大回撤** | -25.57% | **-12.83%** (风险减半) |
| **夏普比率** | 0.72 | **1.23** (极其丝滑) |

**研究结论**：美股的板块动能极强（如 AI 浪潮），不建议用宏观因子干扰选股。**“选股交给动量，风控交给宏观”** 是美股 ETF 轮动的终极答案。通过管住总仓位，我们成功在 2022 年加息大跌中保住了利润。

# A股板块ETF轮动分析系统

## 系统概述

自动采集A股主流板块ETF行情数据，通过多因子量化模型计算综合评分，生成板块轮动信号（强势/中性/弱势），以可视化看板呈现结果。

## 数据流

```
请求 → st.cache_data(内存, TTL=1h)
     → Parquet文件缓存(TTL=4h)
     → akshare API (Sina优先, 东方财富备用)
```

每次请求间隔 0.3s 防封IP。

## 量化模型

### 五大因子

| 因子 | 说明 | 权重 |
|------|------|------|
| 5日动量 | 近5个交易日涨幅 | 0.15 |
| 10日动量 | 近10个交易日涨幅 | 0.15 |
| 20日动量 | 近20个交易日涨幅 | 0.20 |
| RPS相对强弱 | 相对沪深300ETF的强弱比值 | 0.25 |
| 资金流向 | 成交额 vs N日均值比值 | 0.15 |
| 波动率 | 收益率滚动标准差（反向，低波高分） | 0.10 |

### 评分流程

1. 对每只ETF计算上述5个因子的原始值
2. 截面排名归一化：同一时点所有ETF按因子值排名，映射到 0-100 分
3. 加权求和得到综合评分
4. 根据评分生成信号：≥70 强势-关注 / 40-70 中性-观望 / <40 弱势-回避

## ETF覆盖范围（20只）

半导体、新能源、医药、消费、银行、军工、计算机、光伏、农业、房地产、证券、稀土、旅游、有色金属、5G、酒、游戏、传媒、环保、医疗

基准：沪深300ETF（510300）

## 页面说明

### 主看板 (dashboard)
- 信号摘要：强势/中性/弱势各多少只
- 综合评分排行表：按评分降序，含各维度动量值
- 评分排行柱状图
- 评分趋势折线图（Top N）
- 板块轮动热力图（历史评分矩阵）

### 单板块详情 (detail)
- 选择单只ETF查看
- 6个核心指标卡片
- K线+成交量图
- 动量/RPS/资金流/波动率 四个子图

### 参数配置 (settings)
- 调整各因子权重（实时校验总和=1）
- 修改信号阈值
- 查看/管理ETF池
- 清理缓存

# A股板块ETF轮动分析系统

## 系统概述

自动采集A股主流板块ETF行情数据，通过多因子量化模型计算综合评分，生成板块轮动信号（强势/中性/弱势），以可视化看板呈现结果。

## 数据流

```
请求 → st.cache_data(内存, TTL=1h)
     → Parquet文件缓存(TTL=4h)
     → akshare API (Sina优先, 东方财富备用)
```

每次请求间隔 0.3s 防封IP。

## 当前回测口径
*统一假设：T日收盘生成信号，T+1日开盘执行；手续费万0.5；单边滑点 0.05%；训练集 60%，测试集 40%。*

### 三策略定义
- **V1 纯技术动能**：仅使用技术指标打分，不叠加宏观乘数。
- **V2 美债10Y单因子**：基于美国10年期国债收益率方向，对成长板块进行顺风加权/逆风抑制。
- **V3 美债10Y + 沪深300趋势双因子**：在 V2 基础上，再根据沪深300趋势调节蓝筹/防御方向权重。

### 测试集表现（近期样本外）
*区间：2024-02-28 至 2026-03-20，共 499 个交易日。以下“年化”是该测试区间的折算值，不代表长期可持续收益。*

| 策略 | 累计收益 | 折算年化 | 夏普比率 | 最大回撤 |
| :--- | :--- | :--- | :--- | :--- |
| **V1 纯技术动能** | **+155.43%** | **60.57%** | **1.21** | **-19.57%** |
| **V2 美债10Y单因子** | +40.18% | 18.60% | 0.79 | -25.58% |
| **V3 美债10Y + 沪深300趋势** | +52.74% | 23.85% | 0.93 | -20.98% |

### 全样本表现（更接近长期参考）
*区间：2021-01-25 至 2026-03-20，共 1246 个交易日。以下“年化”更适合作为长期能力参考。*

| 策略 | 累计收益 | 折算年化 | 夏普比率 | 最大回撤 |
| :--- | :--- | :--- | :--- | :--- |
| V1 纯技术动能 | +38.37% | 6.79% | 0.34 | -58.82% |
| V2 美债10Y单因子 | +89.54% | 13.81% | 0.52 | -46.97% |
| **V3 美债10Y + 沪深300趋势** | **+124.69%** | **17.79%** | **0.75** | **-35.01%** |

**研究结论**：
1. **逻辑闭环**：当前回测严格执行 T 日信号、T+1 日开盘成交，规避了“收盘信号当天成交”的未来函数错误。
2. **近期爆发不等于长期能力**：测试集折算年化可以很高，但只能代表该阶段行情适配度，不能直接当作长期收益预期。
3. **长期参考以全样本为主**：从 5 年左右样本看，V3 的风险收益比最平衡，V2 次之，V1 波动最大。
4. **实盘预期应保守理解**：A股这套系统更合理的长期年化预期，应按 15% - 20% 左右理解，而不是直接采用单段样本外的高年化结果。

## 最终实盘版

当前已经收敛到一套固定的 V3 双因子版本，不再继续分叉做额外优化：

- **名称**：V3 Final
- **持仓数**：Top2
- **调仓频率**：8 个交易日
- **宏观因子**：美债10Y + 沪深300趋势
- **执行口径**：T 日收盘出信号，T+1 日开盘调仓

固定参数如下：

- `growth_up = 1.5721425692606525`
- `growth_down = 0.5441754551859488`
- `blue_up = 1.1351423122565532`
- `blue_down = 0.9371923292598149`
- `defensive_up = 1.6721000554709928`
- `other_down = 0.5329857361198921`

运行入口：

```bash
python run_final_astock.py
```

这版是当前最接近“可实盘”的固定版本，后续只建议在这个版本上做风控、仓位和交易细节微调，不再继续把研究脚本当主版本。

## 网格策略实验

*说明：这里是独立于轮动策略的网格回测尝试，网格引擎按日线 OHLC 近似日内路径，属于“可交易性验证”，不等同于实盘高频撮合。*

### 近期 5 年实测
*回测区间：2021-03-08 至 2026-03-20*

| 策略 | 训练集年化 | 测试集年化 | 全样本年化 | 最大回撤 |
| :--- | :--- | :--- | :--- | :--- |
| **T+0 低波网格**（债券/黄金/跨境/货币） | -9.25% | 2.78% | -4.60% | -28.84% |
| **板块ETF趋势网格** | -14.07% | 22.47% | -0.89% | -41.37% |

**阶段性结论**：
1. 低波 T+0 网格在当前参数下没有跑出稳定正收益，说明单纯靠波动吃差价，容易被成本和缓慢趋势吞掉。
2. 板块 ETF 趋势网格在测试集里能赚钱，但全样本接近打平，说明它对行情阶段非常敏感，稳定性还不够。
3. 这版网格更适合作为“策略候选原型”，还不能直接当作可上线的主策略。

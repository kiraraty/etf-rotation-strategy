# A股板块ETF轮动分析系统

## 系统概述

自动采集A股主流板块ETF行情数据，通过多因子量化模型计算综合评分，生成板块轮动信号（强势/中性/弱势），以可视化看板呈现结果。

## 数据流

```
请求 → st.cache_data(内存, TTL=1h)
     → Parquet文件缓存(TTL=4h)
     → akshare API (Sina优先, 东方财富备用)
```

每次请求间隔 0.3s 防封IP。

## 量化模型：技术定量 + 宏观定性 (Macro Overlay)

### 核心逻辑：双因子宏观环境判定 (Non-look-ahead)
系统不再仅仅依赖技术指标，而是引入了**“宏观雷达”**来决定赛道偏好：

1.  **流动性因子 (Liquidity)**: 监测 **10年期美债收益率 (US10Y)**。
    - `US10Y < MA20`: 环境变暖，赋予**成长板块** (半导体、科技、新能源) **1.5x 权重乘数**。
2.  **资金流因子 (Capital)**: 监测 **沪深300指数趋势**。
    - `Price > MA20`: 存量/增量资金入场，赋予**蓝筹权重** (白酒、消费、大盘) **1.3x 权重乘数**。
3.  **极端防御模式**: 当利率上行且大盘跌破均线时，强制转向**红利/银行**板块，并压低其余所有板块权重。

### 历史回测表现 (2024.03 - 2026.03)
*测试条件：Top 2 持仓, 6日调仓, 含 0.05% 单边滑点, 万0.5费率*

| 指标 | 纯技术面动能 (Baseline) | **技术+宏观双因子 (Current)** |
| :--- | :--- | :--- |
| **累计收益** | **-12.47%** | **+112.26%** |
| **年化收益** | -6.5% | **45.8%** |
| **最大回撤** | -35.30% | **-21.24%** |
| **夏普比率** | -0.08 | **0.88** |

**研究结论**：A股近两年属于典型的“宏观驱动”市场。纯技术面在频繁的风格切换中会被“左右打脸”，而宏观引擎通过过滤掉“逆风环境”下的虚假动能，实现了收益的质变。

## 页面说明

### 主看板 (dashboard)
- 信号摘要：强势/中性/弱势各多少只
- 综合评分排行表：按评分降序，含各维度动量值
- 评分排行柱状图
- 评分趋势折线图（Top N）
- 板块轮动热力图（历史评分矩阵）

### 单板块详情 (detail)
- 选择单只ETF查看
- 6个核心指标卡片
- K线+成交量图
- 动量/RPS/资金流/波动率 四个子图

### 参数配置 (settings)
- 调整各因子权重（实时校验总和=1）
- 修改信号阈值
- 查看/管理ETF池
- 清理缓存
